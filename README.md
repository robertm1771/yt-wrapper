# yt-wrapper
Simple three-panel YouTube wrapper for streaming or monitoring.

## How it works

- Paste any YouTube watch/live/youtu.be URL in the left input and click **Load YouTube**. The app converts it to an embed-friendly `youtube-nocookie.com` link when possible.
- Enter any two URLs for the right-hand frames and click **Load Right Frames** to display them. Leave inputs blank to clear the frames.
- Press **Enter** inside any input to trigger the appropriate load action.

Status messages below the controls confirm when frames are loaded or cleared.
